### Opening connected symbols

A symbol bubble is the most important bubble of JSBubbles.

JSBubbles wants to make reading code easy as possible without switching between files. Each symbol bubble will have highlighted clickable markers in them. Clicking them will open the relevant symbol.

### Locating the actual symbol

If you click on the `Open file` icon on a symbol bubble's header, it will open the actual file in VSCode.
