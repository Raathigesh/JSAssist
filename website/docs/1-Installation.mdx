### Get it from the marketplace

Install the extension from [marketplace](https://marketplace.visualstudio.com/items?itemName=Raathigeshan.js-bubbles)

### Activating the extension

#### Open the extension through command panel

Open VSCode command panel by pressing `Ctrl+Shift+P` and execute the command `Toggle js bubbles panel`

#### Open by clicking the icon

Click on the icon <img height="35px" alt="Trigger icon" src={useBaseUrl('img/Trigger icon.png')} /> on the top right corner of any file to open the JS Bubbles panel.
